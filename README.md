# Employee Management System

This workspace is split into:

- `frontend` - React application UI
- `backend` - Node.js, Express, MongoDB, and Mongoose API

## Run the project

1. Start MongoDB locally
2. Install backend packages from `backend` with `npm install`
3. Seed MongoDB from `backend` with `npm run seed`
4. Start the backend from `backend` with `npm start`
5. Start the frontend from `frontend` with `npm start`
